## Create a new Astro project

1. Run the following command:

```bash
npm create astro@latest
```

## Add the library

1. Run the following command:

<InstallLibrary />

2. Add the [`Toaster`](/toaster) to the `layout.astro` file:

> ðŸ’¡ Add [`client:load`](https://docs.astro.build/es/reference/directives-reference/#clientload) directive to the `Toaster` component.

```js
// ðŸ“„ layouts/Layout.astro

---
interface Props {
  title: string;
}

import { Toaster } from '@pheralb/toast';

const { title } = Astro.props;
---

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="description" content="Astro description" />
    <meta name="viewport" content="width=device-width" />
    <link rel="icon" type="image/svg+xml" href="/favicon.svg" />
    <meta name="generator" content="{Astro.generator}" />
    <title>{title}</title>
  </head>
  <body>
    <slot />
    <Toaster client:load />
  </body>
</html>
```

4. Import the `ShowToast` component in the `index.astro` file:

> ðŸ’¡ Add [`client:load`](https://docs.astro.build/es/reference/directives-reference/#clientload) directive to the `ShowToast` component.

```js
  </main>

```
